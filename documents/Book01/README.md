# 人生攻略第一部-三观重塑

## 前言

1. 我是一个很喜欢简洁与格式化的人，本书与后续系列你都会看到很多比如：人类应该少吃东西，但我不会告诉你为什么，如果你感兴趣，可以询问Chatgpt。我鼓励你去查询，我认为好奇与勇敢尝试是人类的第一品格。
2. 信息密度较大，因为是基于我的读书笔记转化而来的，我读书笔记是条状的。
3. 接受世界的主观性。
4. 这种类型的书也不好写，我想可以写成问答类的，我也认为提问是最能识别三观的方式。

我觉得人生只有重要的三件事

1. 要挣很多很多钱。
2. 要活的很久很久。
3. 要找到一个很爱很爱的人。

你需要成为一个什么样的人？

1. 我以后也要成为像爸爸一样的人。
2. 我这辈子做过的最正确的决定就是嫁给他。
3. 他是一个非常值得交往的朋友。
4. 你儿子真是孝顺啊。

动物保护法？牛牛就不保护了？也很可爱啊，小🐏就不保护了吗？就只是因为猫猫狗狗为人类提供情绪价值，就忽略了人类漫长发展史中对人类生产力提供帮助的牛马们吗？最重要的事情就是，又要增设政府部门增加冗余的官员，这笔钱谁出？拿出来给山区的孩子改善伙食不好吗？
