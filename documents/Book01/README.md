# 人生攻略第一部-我的三观（其实是吐槽合集）

## 前言

1. 和本系列的后续书籍相比，这本更倾向于自我表达，可看可不看，可以直接跳过去看第二部也没有问题。
2. 也许我描述的这个世界和你的理解有些偏差，但请阁下认识到这个世界的主观性。
3. 这种类型的书也不好写，我想可以写成问答类的，我也认为提问是最能识别三观的方式，算了想到什么写什么。

我觉得人生只有重要的三件事

1. 要挣很多很多钱。
2. 要活的很久很久。
3. 要找到一个很爱很爱的人。

你需要成为一个什么样的人？

1. 爸爸，你是我最想成为的人。
2. 我这辈子做过的最正确的决定就是嫁给他。
3. 他是一个非常值得交往的朋友。
4. 你儿子真是孝顺啊。

## 后记

1. 我确实写了一部分，但是我后来都删掉了，这和我的生存法则不匹配。
2. 其实和每个人的成长经历有关，我害怕得罪人，即使我的地位远高于他，我仍旧害怕去与他产生冲突。
3. 我是这样一个人：你夸xxx即使我非常讨厌xxx，我仍旧会认同你。你说xxx是xxx的特产，我知道你说错了，我仍旧会对你讲，是这样吗？又涨知识了，跟着你就是能学到东西。
4. 我的人际交往策略永远都是：“规避冲突”。
5. 与人交谈时我永远会选择不基于事实、信仰或者逻辑，对于我们的对话，我更关注与我们之间的关系。
6. 我写了大量的不符合国内主流价值观的东西，但我认为我写的东西是被国际认同的，但并不会被国内观众认同，所以我思考了几分钟，最后还是删掉了。
