## 今日检查清单

- [ ] 远离吸烟者
- [ ] 不可食用 smoked(烟熏) 和腌制肉类，因其可能增加某些肿瘤风险。
- [ ] 感受 hungry(饥饿)
- [ ] 禁止精制糖与代糖养成低甜阈值
- [ ] 冷暴露（对比浴/冷水收尾/低温散步）
- [ ] 总量控制水果，果糖全年大量摄入会持续冲击肾脏
- [ ] 阅读配料表：若前排是糖/氢化油/奶精/植脂末/麦芽糖浆/反式脂肪谨慎购买。
- [ ] 减少人工色素
- [ ] 警惕“营养密度低” 能量足但营养素匮乏；优先选择营养密度高的食物。
- [ ] 睡眠环境保持“黑、静、凉”：更凉爽更利于入睡（通常≈18℃）。
- [ ] 规律运动，但避免睡前 2–3 小时进行剧烈训练。
- [ ] 人们无论如何不应该进行高强度训练，而应该进行柔和训练！
- [ ] 用很小的餐盘，允许剩菜。
- [ ] 超加工食品的饮食。什么叫超加工食品，就是你已经看不出食物原来的样子了。要减少食用这些过度加工的食品，它们会扰乱微生物组，原因尚不完全清楚。
- [ ] 过量摄入红肉会在体内引起轻度炎症，影响生物组的平衡。
- [ ] 不要一心多用，在吃饭的时候，如果你在旁边放个手机，一边刷手机一边吃饭，这对胃和大脑都不好。
- [ ] 一旦你没事干了，你的脑子很快就退化了。所以要给自己找点事干，哪怕你养只小鸡，给自己找一点责任。经常学新东西，这时候你的大脑才能够更健康。
- [ ] 很多人为了保护膝盖不运动，这其实是非常不合理的，你会发现越运动的人膝盖越好。
- [ ] “你的身体不假思索地自动校准或自我纠正的能力越强，你受伤的风险就越低。”简单的平衡练习可以提高你的本体感觉。闭上眼睛，单腿微微弯曲站立。这个动作不仅会迫使膝盖周围的肌肉在保持你的平衡时付出更多的努力，还能调整你的关节感觉。你还可以把头偏向一侧，以提高难度。……你定期参加越多的体育活动，你的身体和大脑就越有可能本能地知道该做什么。
- [ ] 肠道里的细菌还喜欢吃一种叫作多酚的分子，蓝莓、特级初榨橄榄油、黑巧克力和咖啡中都有这种分子。这些食物能有效地帮助健康的肠道生物群落保持良好状态。
- [ ] 即使没有阳光，你也需要用防晒霜，这能够降低皮肤癌的风险，还能防止晒伤。更重要的是，防晒霜是真正抗衰老的产品。因为衰老主要是氧化和糖化引起的，你晒得多就容易氧化。防晒系数（SPF）是指防晒霜可以保护皮肤免受中波紫外线（UVB）伤害的时间，时间越长，这个数值就越高。SPF30可以阻挡97%的中波紫外线。
- [ ] 姜有抵抗恶心的功效。那晕车感觉恶心是怎么回事呢？晕车就是你的眼睛和平衡传感器不匹配了，你的大脑会认为你中毒了，所以你会想要赶紧吐。这时候怎么办呢？关键是要看向地平线。就是你要坐在车的最前面，尽量地看远处的地平线，不要看路两边的树。如果你看着路两边的树哗哗地动，或者低头看手机，都是特别容易晕车的。看着远方的地平线，你就会感觉好一些。当然经过长时间的锻炼，坐车坐的时间久了，你的大脑会慢慢告诉你没关系，不要紧。

如果你能够学会避免屏气用力排便，就会降低患痔疮的风险。就是说大便的时候不能使太大劲，使太大劲可能会造成痔疮。大便的时间最好限制在10分钟以内，如果可以，最好不要超过5分钟。

就是冲厕所的时候一定要盖马桶盖。为什么这很重要呢？很多家庭的厕所和洗脸池是在一起的，而当抽水马桶的水以很快的速度冲下来的时候，粪便的颗粒会以每秒两米的速度喷到空中，所以空气中会出现大量粪便的颗粒。而那些粪便的颗粒里边含有各种细菌，如果通过你的口腔、眼睛进入身体里，体内就会产生炎症。





每个人都天生具有一种古老的特征，叫作哺乳动物潜水反射。当你屏住呼吸将脸浸入冷水时，它会被触发。……一旦你的脸进入水下，鼻孔里充满了水，这个信息就会通过三叉神经（第5对脑神经）传递给大脑，进而导致迷走神经（第10对脑神经）诱发心动过缓。这会促使血管变细，限制血液流向四肢，以便向心脏、大脑和肺部供血。这种生理变化可以减少焦虑和压力。这是哺乳动物的一种天然的反应。









怎么做才能对眼睛健康有帮助呢？有一个方法叫作20-20-20法则，这个方法对我们的视力保护是有帮助的。“每隔20分钟，离开屏幕休息一下，看向至少20英尺（约为6.096米）外的物体，至少看20秒。”这叫20-20-20法则，有助于保护视力。



流鼻血时，大家一般是怎么止血的？我们在小时候流了鼻血，一般的做法是不是头抬起来，举一个手，哪边的鼻孔流血就举另外一边的手。作者说正好相反，当你流鼻血的时候，把头抬起来是很危险的，因为血液会流到你的口腔里，幸运的话只是流到胃里，可能会造成拉肚子，因为血液是很好的泻药。如果不小心流到气管里，可能会造成生命危险。而且抬头并不能够保证让你的鼻血停止。

他说当你流鼻血的时候，最好的方法是什么呢？就是捏住鼻子上的这个软的地方，然后坐直，把头稍微向前倾，以阻止血液流到喉咙后部。如果这个样子鼻血还停不下来，就用一块小纸巾折成一个小方块，垫在牙龈和上唇之间，这个动作可以对鼻子里的血管加压。另外，简单的冰敷也是一个办法，冰敷可以使得鼻子附近的血管收缩。用这几个办法就能够让鼻血止住。
