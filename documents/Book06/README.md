# 人生攻略第六部-如何才能活得更久？

## 我的可复制的一天

- [ ] 北京时间02：00睡觉到当日9：00起床（遵循R90睡眠法）
- [ ] 生姜片放在嘴里慢慢咀嚼
- [ ] 早上食用南瓜子25克
- [ ] 枸杞干嚼20克
- [ ] 早起一杯黑咖啡（什么都不能加，不能放太久需要立刻喝）
- [ ] 把昨天晚上泡好的豆子放进豆浆机，放到60度的时候兑点纯牛奶。
- [ ] 洗漱
- [ ] 烧水壶里面放1L清水比1g食用盐，滴两滴橄榄油，放点你切好的蔬菜比如西兰花，莲藕，豆皮，猪肉。（最好是养生壶可以放办公室里面的那种，，我一开始觉得喝水喝吃饭用同一个器具有点犯恶心，但是如果我把他当作我喝了一天的汤，我就能接受了）
- [ ] 9：15上班（建议大家下载，每个你设定的时间都会强制锁电脑，运动拉伸）
- [ ] 上班过程中捞干净壶里面的食材，想倒就倒（我一天能喝5，6斤水我不爱倒），也可以在汤里面直接放茶叶什么的
- [ ] 可可≥70%的黑巧克力每日小份≈25g
- [ ] 10：00带着笔记本电脑去超市买明天的食材，绕弯走一万步（务必少买，这样每天都能逛超市了）





- [ ] 晚上食用南瓜子25克
- [ ] 睡前检查: 有无喝2L水
- [ ] 睡前检查: 8+16禁食法
- [ ] 睡前检查: 用横着走（弯点膝盖）倒着走替代外出时前进的方向




- [ ] 睡前检查: 拉伸



- [ ] 低脂牛奶或酸奶


- [ ] 原地踮脚尖（不要行走，就原地）
- [ ] 每日30–45分钟70%心率有氧运动
- [ ] 力量训练：深蹲 / 髋铰链 / 推 / 拉 / 提 / 握力
- [ ] 远离吸烟者
- [ ] 感受 hungry(饥饿)
- [ ] 精制糖与含糖饮料“限得更狠”无糖/低糖饮品
- [ ] 常备坚果与种子作加餐
- [ ] 冷暴露（对比浴/冷水收尾/低温散步）
- [ ] 总量控制水果，果糖全年大量摄入会持续冲击肾脏
- [ ] 如需小睡，限制在20–30分钟且不要太晚。
- [ ] 午后不再摄入咖啡因（至少睡前8–10小时停用）。
- [ ] 阅读配料表：若前排是糖/氢化油/奶精/植脂末/麦芽糖浆/反式脂肪谨慎购买。
- [ ] 减少人工甜味剂与人工色素
- [ ] 脾胃喜欢细碎的 food(食物) 尽量一口 30-60 咀嚼下
- [ ] 警惕“空能量食物” 能量足但营养素匮乏；优先选择营养密度高的食物。
- [ ] 代糖“能用但别滥用”：甜叶菊/罗汉果等高强度代糖能过渡口味，但不等于无限量；养成低甜阈值才是根本。
- [ ] 打破久坐：每小时起来动一动，动则有益。
- [ ] 推荐天天吃水果，每天摄入 200–350g 新鲜水果，果汁不能代替鲜果。
- [ ] 睡眠环境保持“黑、静、凉”：更凉爽更利于入睡（通常≈18℃）。
- [ ] 规律运动，但避免睡前 2–3 小时进行剧烈训练。
- [ ] 饭后别久坐：步行 20 分钟或做下台阶练习
- [ ] 下午后不再摄入咖啡因（至少睡前 8–10 小时停用）。
- [ ] 不可食用 smoked(烟熏) 和腌制肉类，因其可能增加某些肿瘤风险。
- [ ] 早晨推迟第一餐，直到“真正饥饿”再吃
- [ ] “慢吃＋专注吃”：每餐至少 20 分钟，七分饱停
- [ ] 耐力运动者的心跳频率相较正常人偏低，意味着心脏负担更小。
- [ ] 人们无论如何不应该进行高强度训练，而应该进行柔和训练！
- [ ] 吃饭：餐前 20 分钟内饮用温醋水300 ml，清淡饮食习惯，用很小的餐盘，允许剩菜。
