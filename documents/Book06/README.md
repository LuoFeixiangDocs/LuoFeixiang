# 人生攻略第六部-如何才能活得更久？

## 每日必做

- [ ] 早起一杯黑咖啡（什么都不能加，不能放太久立马喝）
- [ ] 拉伸
- [ ] 每日喝2L水
- [ ] 8+16禁食法
- [ ] 枸杞干嚼20克
- [ ] 低脂牛奶或酸奶
- [ ] 早晚各食用南瓜子25克
- [ ] 每天步行 10,000 步
- [ ] 生姜片放在嘴里慢慢咀嚼
- [ ] 横着走（弯点膝盖）倒着走
- [ ] 原地踮脚尖（不要行走，就原地）
- [ ] 每日30–45分钟70%心率有氧运动
- [ ] 可可≥70%的黑巧克力每日小份≈25g
- [ ] 力量训练：深蹲 / 髋铰链 / 推 / 拉 / 提 / 握力

## 好的习惯

- [ ] 远离吸烟者
- [ ] 培养清淡饮食习惯
- [ ] 感受 hungry(饥饿)
- [ ] 反式脂肪 美国已在食品生产中禁止使用。
- [ ] 精制糖与含糖饮料“限得更狠”
- [ ] 减盐、减油、减糖； 淡口味、少油烹、无糖/低糖饮品；
- [ ] 少用冷饮、久吹空调、露腹脐；注意腹部保暖。
- [ ] 打破“清盘文化”：允许餐盘里留食物
- [ ] 进食行为：小盘策略
- [ ] 常备坚果与种子作加餐
- [ ] 冷暴露（对比浴/冷水收尾/低温散步）
- [ ] 总量控制水果，果糖全年大量摄入会持续冲击肾脏
- [ ] 如需小睡，限制在20–30分钟且不要太晚。
- [ ] 午后不再摄入咖啡因（至少睡前8–10小时停用）。
- [ ] 阅读配料表：若前排是糖/氢化油/奶精/植脂末/麦芽糖浆，谨慎购买。
- [ ] 减少人工甜味剂与人工色素
- [ ] 脾胃喜欢细碎的 food(食物) 尽量一口 30-60 咀嚼下
- [ ] 警惕“空能量食物” 能量足但营养素匮乏；优先选择营养密度高的食物。
- [ ] 代糖“能用但别滥用”：甜叶菊/罗汉果等高强度代糖能过渡口味，但不等于无限量；养成低甜阈值才是根本。
- [ ] 打破久坐：每小时起来动一动，动则有益。
- [ ] 推荐天天吃水果，每天摄入 200–350g 新鲜水果，果汁不能代替鲜果。
- [ ] 睡眠环境保持“黑、静、凉”：更凉爽更利于入睡（通常≈18℃）。
- [ ] 规律运动，但避免睡前 2–3 小时进行剧烈训练。
- [ ] 饭后别久坐：步行 20 分钟或做下台阶练习
- [ ] 下午后不再摄入咖啡因（至少睡前 8–10 小时停用）。
- [ ] 不可食用 smoked(烟熏) 和腌制肉类，因其可能增加某些肿瘤风险。
- [ ] 早晨推迟第一餐，直到“真正饥饿”再吃
- [ ] “慢吃＋专注吃”：每餐至少 20 分钟，七分饱停
- [ ] 耐力运动者的心跳频率相较正常人偏低，意味着心脏负担更小。
- [ ] 人们无论如何不应该进行高强度训练，而应该进行柔和训练！
- [ ] 进食行为：饭前 10–20 分钟喝 200–300 ml 温水，减少过量进食。
- [ ] 餐前 20 分钟或餐中/餐后 20 分钟内喝完；可在吃前饮用“醋水”（1 汤匙兑一大杯水）。
