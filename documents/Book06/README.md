# 人生攻略第六部-如何才能活得更久？

- [ ] 早起一杯黑咖啡（什么都不能加，不能放太久立马喝）
- [ ] 拉伸
- [ ] 每日喝2L水
- [ ] 8+16禁食法
- [ ] 枸杞干嚼20克
- [ ] 低脂牛奶或酸奶
- [ ] 早晚各食用南瓜子25克

- [ ] 生姜片放在嘴里慢慢咀嚼
- [ ] 横着走（弯点膝盖）倒着走
- [ ] 原地踮脚尖（不要行走，就原地）

- [ ] 可可≥70%的黑巧克力每日小份≈25g
